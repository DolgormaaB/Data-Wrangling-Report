# Data Wrangling Report - DDS

Welcome to our group's Data Wrangling report! This repository contains our solutions for the three exercises assigned for Lab 3, designed to practice data manipulation using the Pandas library in Python.

Below is a summary of the exercises we completed and links to the corresponding code.

---

### 📂 [Exercise 1: Occupation Data](./Exercise_1)
**Topic:** Basic Pandas Data Exploration
**Summary:** In this exercise, we explored an "Occupations" dataset. We practiced reading data into a Pandas DataFrame, understanding the basic structure of the data (rows, columns, data types), and performing basic demographic analysis based on the different occupations represented.

---

### 📓 [Exercise 2: Euro 2012 (Filtering & Sorting)](./Ex2.ipynb)
**Topic:** Filtering and Sorting Data
**Summary:** This exercise focused on extracting specific information from a dataset containing statistics from the UEFA Euro 2012 soccer tournament. We applied Pandas filtering techniques to isolate teams based on specific conditions (like goals scored) and practiced sorting the DataFrame to organize our findings.

---

### 📓 [Exercise 3: Alcohol Consumption (Grouping)](./Ex3.ipynb)
**Topic:** Data Grouping and Aggregation
**Summary:** Using an Alcohol Consumption dataset, we practiced the powerful `groupby()` function in Pandas. We grouped the data by continent to calculate and compare aggregate statistics (like mean and max consumption of different types of alcohol) across different regions of the world.

---

### Conclusion
Through these three exercises, our group successfully practiced essential data wrangling techniques, including:
* Importing and inspecting datasets.
* Filtering rows based on conditional logic.
* Sorting DataFrames by one or multiple columns.
* Grouping data and applying aggregation functions to draw insights.
